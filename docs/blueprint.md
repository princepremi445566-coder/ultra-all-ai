# **App Name**: Ultra All AI

## Core Features:

- Secure User Authentication: Implement Firebase Authentication allowing users to sign up/in via Email & Password, Google, or Phone with OTP, including a secure logout system and real-time auth state listening.
- Protected AI Tools Dashboard: A central dashboard accessible only to authenticated users, serving as the gateway to all AI and utility tools.
- AI Tool: Instagram Bio Generator: A generative AI tool that creates compelling Instagram bios based on user-provided keywords, tone preferences, and context.
- AI Tool: YouTube Title Generator: A generative AI tool that suggests optimized YouTube video titles based on video description, keywords, and target audience.
- Utility Tool: QR Code Generator: Allow users to generate QR codes for any provided URL or text input.
- Tool Usage History & Results Storage: Store a record of each user's tool usage history and their generated outputs within Firestore for easy retrieval.
- User Profile Management: Display user-specific profile data from Firebase and allow for basic settings adjustments.

## Style Guidelines:

- A light color scheme emphasizing clarity and premium feel. Primary color: #814AB8 (a deep violet, chosen for its modern and sophisticated, yet vibrant quality). Background color: #F4F0F8 (a highly desaturated, light purple, creating a clean canvas).
- Accent color: #668FED (a bright, analogous blue that provides dynamic contrast and complements the purple, suggesting a 'purple to blue' gradient accent).
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and neutral aesthetic, suitable for a clean and professional SaaS startup look.
- Utilize minimalist, geometric line icons that align with a modern glassmorphism design, providing clarity without visual clutter.
- Implement a mobile-first responsive layout with ample whitespace to achieve a clean and premium user interface, focusing on intuitive navigation.
- Incorporate subtle and smooth animations and transitions for element interactions, loading states, and view changes to enhance user experience and convey a fast, polished feel.
